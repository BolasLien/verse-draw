# Verse Draw

一個安靜的經文抽卡小工具。向下拖曳卡片後，畫面會播放翻卡動畫，並從 Bible API 取得一段中文和合本經文。

## Features

- React + Vite
- CSS card reveal animation
- Pointer events drag interaction
- Random Chinese Bible verse from `https://bible-api.com/data/cuv/random`
- Local fallback verse when the API is unavailable
- GitHub Pages deployment workflow

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run test
npm run build
```
